# Ecommerce WebApp (React) 🛒

## 📌 About the Project

This is a full-stack eCommerce web application similar to Amazon or Flipkart. Users can browse products, add them to cart, place orders, and track their purchases.

The project focuses on building a scalable system by separating frontend, backend, and database, and deploying them on AWS services.

---

## 🚀 Features

* Browse and search products using keywords
* Add products to cart
* Update or remove items from cart
* Choose delivery preferences (like delivery date)
* Place orders
* View order history
* Track order status
* Full CRUD operations (create, update, delete products/orders)

---

## 🛠️ Tech Stack

* Frontend: React, JavaScript, HTML, CSS
* Backend: Node.js, Express
* Database: SQL (AWS RDS)
* Deployment: AWS (Elastic Beanstalk, EC2, S3)

---

## 🌐 Deployment Details

* Frontend and backend are deployed separately using AWS Elastic Beanstalk
* Database is hosted on AWS SQL service (RDS)
* Used AWS S3 for storage
* Configured load balancing and routing (Route 53)
* SSL setup was implemented for HTTPS (later removed due to free tier limits)
